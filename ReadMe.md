# 📚 Sistema de Cadastro de Autores e Livros (Django)

## 📌 Descrição

Este projeto é um sistema web simples desenvolvido com **Django** para realizar o **gerenciamento de autores e livros**.

O sistema permite executar operações de **CRUD (Create, Read, Update, Delete)** para autores e livros.

Cada livro está associado a um autor, estabelecendo uma **relação entre as entidades no banco de dados**.

O objetivo do projeto é demonstrar a construção de uma aplicação web utilizando **Django, ORM, templates e formulários HTML**.

---

# ⚙️ Tecnologias utilizadas

- Python
- Django
- SQLite
- HTML
- Django Templates
